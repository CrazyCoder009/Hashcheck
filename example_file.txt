This is a sample file where you can modify.

Everytime you modify This file a new md5hash is generated for it.

You can use this script to check for md5hash of any file.

How to run the script:

bash icheck.sh [filename] [The md5hash you need to check against]

eg: 
bash icheck.sh example_file.txt 9e5ff423ca31c4e4236f137d1081ff1a
